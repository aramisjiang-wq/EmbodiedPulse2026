```

├── README.md              # 本文档（文档目录说明）
├── index.md               # 文档首页（如使用GitHub Pages）
```

## 📚 文档分类

### 项目文档（docs/项目文档/）

所有详细的项目文档都在这里，按以下分类组织：

- **产品文档/** - 产品介绍、设计理念、开发分享
- **需求文档/** - PRD、需求分析
- **技术文档/** - 技术架构、数据库、部署
- **功能说明/** - 功能详细说明（按模块分类）
- **使用说明/** - 用户使用指南
- **优化记录/** - UI/UX优化和性能优化记录
- **问题修复/** - Bug修复记录
- **测试文档/** - 测试结果和功能检查
- **开发记录/** - 开发过程记录、复盘、建议
- **开发规范/** - 开发原则、规范、最佳实践
- **沟通记录/** - 开发沟通历程

**详细索引**：查看 [项目文档/README.md](./项目文档/README.md)
