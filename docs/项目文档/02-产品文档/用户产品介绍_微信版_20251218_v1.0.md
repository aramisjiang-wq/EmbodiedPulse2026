## 🚀 核心功能

### 具身论文页面

1. **研究方向活跃度模块**
   - 时间范围选择：最近4周 / 最近8周
   - 视图切换：分类视图 / 子标签视图
   - 交互式图表：点击图例筛选，悬停查看详情

2. **论文清单**
   - 33个分类标签，点击查看对应论文
   - 支持按日期、引用数、标题排序
   - 支持按发表场所筛选（ArXiv、会议、期刊等）

3. **搜索功能**
   - 实时搜索：输入关键词立即显示结果
   - 全文搜索：标题、作者、摘要全覆盖
   - 高亮显示：搜索结果中关键词高亮

4. **右侧挂件**
   - 论文统计：总数、今日、本周、本月新增
   - 活跃作者排行榜：TOP作者及其研究方向
   - 其他信息：B站信息、岗位机会、数据集

### 具身视频Hub页面

1. **视频发布趋势**
   - 堆叠面积图：展示各公司视频发布数量趋势
   - 热力图：按公司、按日期展示发布密度
   - 图例交互：点击图例筛选公司

2. **月度播放量对比**
   - 近1年热力图：各公司月度播放量对比
   - 颜色深浅：直观显示播放量高低

3. **公司卡片列表**
   - 12个UP主信息卡片
   - 每个UP主的最新视频列表
   - 播放量、评论数、收藏数统计

4. **右侧挂件**
   - 7天内最新视频：按发布时间倒序
   - TOP 5排行：近30天发布数量最多的公司
   - 年度总播放量：当年各公司累计播放量对比

## 📋 数据更新规则

### 论文数据
- **自动更新**：每小时整点自动抓取ArXiv最新论文（最近21天）

- **新论文提示**：昨天新增的论文会在"刷新论文数据"按钮旁显示红点

- **前端刷新**：页面每5分钟自动刷新一次
- **API缓存**：10分钟缓存，避免API频率限制
- **手动刷新**：点击"刷新数据"按钮可强制更新（跳过缓存）

## ❓ 常见问题

**Q: 研究方向活跃度图表怎么看？**  
A: 图表展示各研究方向在最近4周/8周的论文发布数量趋势。颜色越深、高度越高，表示该方向越活跃。点击图例可以筛选显示特定方向。

**Q: 活跃作者排行榜多久更新一次？**  
A: 与论文数据同步更新，每小时自动更新一次。排行榜显示发表论文最多的作者及其研究方向。

**Q: 视频数据多久更新一次？**  
A: 后端每6小时自动更新一次，前端页面每5分钟自动刷新一次。你也可以点击"刷新数据"按钮强制更新。

**Q: 如何查看某个公司的所有视频？**  
A: 在"公司列表"中找到对应的UP主卡片，点击查看该UP主的所有视频。

**Q: 如何搜索论文？**  
A: 在搜索框输入关键词（标题、作者、摘要），按Enter或点击搜索按钮。支持实时搜索，输入即显示结果。
