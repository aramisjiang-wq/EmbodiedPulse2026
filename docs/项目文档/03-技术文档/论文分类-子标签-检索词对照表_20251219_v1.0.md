# 论文分类-子标签-检索词对照表

## 说明

本文档包含两个配置文件的完整对照表：
1. **`taxonomy.py`** - 标签分类体系（用于前端显示和分类）
2. **`config.yaml`** - ArXiv 论文抓取配置（用于论文检索）

## 三、两个配置文件的差异说明

### 主要差异

1. **标签键格式**:
   - `taxonomy.py`: 使用 `/` 分隔（如 `Operation/Grasp`）
   - `config.yaml`: 使用 `-` 分隔（如 `Operation-Grasp`）

2. **检索词数量**:
   - `taxonomy.py`: 检索词更详细、更全面（用于精确分类）
   - `config.yaml`: 检索词较少、更通用（用于 ArXiv API 检索）

3. **Benchmark 检索词**:
   - `taxonomy.py`: 只有 `embodied benchmark`（1个）
   - `config.yaml`: 包含 `benchmark, evaluation, dataset, testbed, simulator, simulation`（6个）

### 使用建议

- **`taxonomy.py`**: 用于前端显示、论文分类、标签匹配
- **`config.yaml`**: 用于 ArXiv 论文抓取时的关键词检索
