## 🔌 API接口列表

### 飞书登录API
- `POST /api/auth/feishu/login` - 飞书登录
- `GET /api/auth/feishu/callback` - 飞书登录回调

### 用户信息API
- `GET /api/user-info` - 获取用户信息
- `PUT /api/user/profile` - 更新用户信息

### 登录历史API
- `GET /api/user/login-history` - 获取登录历史
- `GET /api/user/access-logs` - 获取访问记录

### 退出登录API
- `POST /api/logout` - 退出登录
