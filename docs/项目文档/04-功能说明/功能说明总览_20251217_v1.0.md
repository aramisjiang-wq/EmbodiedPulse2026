## 🔌 API接口

了解所有API接口的详细说明，包括请求参数、响应格式、使用示例。

### 总览
- [API接口总览](./API接口/README.md) - 所有API接口的索引

### 论文API
- `GET /api/papers` - 获取论文列表
- `GET /api/paper-stats` - 获取论文统计
- `POST /api/fetch` - 触发论文抓取
- `GET /api/fetch-status` - 获取抓取状态

### 视频API
- `GET /api/bilibili/all` - 获取所有UP主数据
- `GET /api/trends` - 获取视频发布趋势
- `GET /api/research-activity` - 获取研究方向活跃度

### 管理API
- `GET /api/admin/papers` - 获取论文列表（管理端）
- `GET /api/admin/bilibili/ups` - 获取UP主列表（管理端）
- `GET /api/admin/bilibili/videos` - 获取视频列表（管理端）

### 认证API
- `POST /api/auth/feishu/login` - 飞书登录
- `GET /api/user-info` - 获取用户信息
- `POST /api/logout` - 退出登录

### 数据抓取脚本
- `fetch_new_data.py` - 抓取新论文数据
- `fetch_bilibili_data.py` - 抓取B站数据
- `fetch_news.py` - 抓取新闻数据
- `fetch_jobs.py` - 抓取招聘信息

- `check_bilibili_api_to_db.py` - 检查B站API到数据库的更新
- `check_bilibili_update_system.py` - 检查B站数据更新系统
- `diagnose_paper_fetch.py` - 诊断论文抓取问题
- `data_health_check.py` - 数据健康检查

### 数据修复脚本
- `reclassify_all_papers.py` - 重新分类所有论文
- `fix_misclassified_papers.py` - 修复错误分类的论文
- `improve_semantic_update.py` - 改进Semantic Scholar更新

### 部署脚本
- `deploy.sh` - 部署脚本
- `server_auto_deploy.sh` - 服务器自动部署
- `restart_server.sh` - 重启服务器
