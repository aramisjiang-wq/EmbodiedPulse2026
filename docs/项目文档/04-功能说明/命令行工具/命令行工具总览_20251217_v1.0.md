## 🛠️ 命令行工具分类

### 数据抓取脚本
- [数据抓取脚本索引](./数据抓取脚本/README.md)
- `fetch_new_data.py` - 抓取新论文数据
- `fetch_bilibili_data.py` - 抓取B站数据
- `fetch_news.py` - 抓取新闻数据
- `fetch_jobs.py` - 抓取招聘信息

### 数据检查脚本
- [数据检查脚本索引](./数据检查脚本/README.md)
- `check_bilibili_api_to_db.py` - 检查B站API到数据库的更新
- `check_bilibili_update_system.py` - 检查B站数据更新系统
- `diagnose_paper_fetch.py` - 诊断论文抓取问题
- `data_health_check.py` - 数据健康检查

### 数据修复脚本
- [数据修复脚本索引](./数据修复脚本/README.md)
- `reclassify_all_papers.py` - 重新分类所有论文
- `fix_misclassified_papers.py` - 修复错误分类的论文
- `improve_semantic_update.py` - 改进Semantic Scholar更新

### 部署脚本
- [部署脚本索引](./部署脚本/README.md)
- `deploy.sh` - 部署脚本
- `server_auto_deploy.sh` - 服务器自动部署
- `restart_server.sh` - 重启服务器

### 工具脚本
- [工具脚本索引](./工具脚本/README.md)
- `export_papers_to_csv.py` - 导出论文数据为CSV
- `check_duplicates.py` - 检查重复数据
