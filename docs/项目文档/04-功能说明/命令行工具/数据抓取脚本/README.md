## 📥 数据抓取脚本列表

### 论文抓取脚本
- `fetch_new_data.py` - 抓取新论文数据（主要脚本）
- `fetch_december_papers.py` - 抓取12月论文
- `fetch_dec16_papers.py` - 抓取12月16日论文
- `fetch_specific_date.py` - 抓取指定日期范围的论文
- `full_fetch_papers.py` - 完整抓取论文（更广泛的查询）

### 视频抓取脚本
- `fetch_bilibili_data.py` - 抓取B站UP主和视频数据

### 新闻抓取脚本
- `fetch_news.py` - 抓取新闻数据

### 招聘抓取脚本
- `fetch_jobs.py` - 抓取招聘信息
