## 🔧 配置说明

### UID提取规则

从B站空间链接中提取UID：
- 格式1：`https://space.bilibili.com/{UID}` → UID直接提取
- 格式2：`https://space.bilibili.com/{UID}?spm_id_from=...` → UID为问号前的数字

### 配置步骤

1. **添加新UP主**：
   - 在下方"UP主列表"中添加新条目
   - 填写UP主名称、UID、空间链接
   - 更新状态为"✅ 已配置"

2. **更新UP主信息**：
   - 通过B站API获取UP主名称
   - 更新表格中的名称信息

3. **移除UP主**：
   - 将状态改为"❌ 已移除"
   - 保留历史记录
