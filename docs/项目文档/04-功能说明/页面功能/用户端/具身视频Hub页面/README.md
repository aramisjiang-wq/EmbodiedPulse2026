---

## 📋 页面概述

具身视频Hub页面提供B站具身智能领域UP主和视频内容的追踪、数据可视化和统计分析功能。

---

## 🎯 核心功能

### 1. UP主信息展示
- 12个UP主追踪
- 粉丝数、视频数、播放量统计
- UP主信息卡片展示

### 2. 视频发布趋势可视化
- 堆叠面积图（各公司视频发布数量趋势）
- 热力图视图（按公司、按日期展示发布密度）
- 图例交互（点击图例筛选公司）
- 自然周数据维度（周一到周日）

### 3. 播放量数据对比
- 月度播放量对比（近1年热力图）
- 年度总播放量（当年各公司累计播放量）
- TOP 5排行（近30天发布数量最多的公司）

### 4. 最新视频追踪
- 7天内最新视频（按发布时间倒序）
- 视频列表展示（标题、播放量、评论数、收藏数）
- 实时更新（每6小时自动更新，前端5分钟刷新）

### 5. 数据刷新
- 前端自动刷新（5分钟）
- 手动刷新按钮（跳过缓存）
- API缓存机制（10分钟）
