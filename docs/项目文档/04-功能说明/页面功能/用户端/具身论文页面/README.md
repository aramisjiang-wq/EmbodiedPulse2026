## 📋 页面概述

具身论文页面是Embodied Pulse的核心页面，提供具身智能领域论文的浏览、搜索、分类和统计功能。

---

## 🎯 核心功能

### 1. 论文展示
- 33个分类标签，350+检索词
- 按类别分类展示论文
- 支持切换标签页
- 显示论文标题、作者、发布日期、PDF和代码链接

### 2. 研究方向活跃度可视化
- 实时趋势图（最近4周/8周）
- 分类视图（33个方向）
- 子标签视图（细分研究领域）
- 交互式图表（点击图例筛选，悬停查看详情）

### 3. 活跃作者排行榜
- 高产作者追踪
- 研究方向标注
- 论文数量统计

### 4. 搜索和筛选
- 实时搜索（标题、作者、摘要）
- 按发表场所筛选
- 按日期、引用数、标题排序
- 关键词高亮显示

### 5. 数据刷新
- 手动刷新论文数据
- 实时进度显示
- 状态轮询机制
- 新论文红点提示

### 6. 统计信息
- 总论文数统计
- 各类别论文数量
- 今日新增、本周新增、本月新增
- 最后更新时间
