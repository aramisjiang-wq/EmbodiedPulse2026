# 管理平台功能说明

**页面路径**: `/admin/*`  
**访问地址**: https://admin123.gradmotion.com/admin/  
**最后更新**: 2025-12-19

---

## 📋 页面概述

管理平台提供完整的管理功能，包括用户管理、论文管理、视频管理、日志监控和统计仪表盘。

---

## 🎯 核心功能

### 1. 仪表盘
- 数据概览统计
- 图表展示
- 系统配置信息
- 手动更新B站数据功能

### 2. 用户管理
- 查看用户列表
- 编辑用户信息
- 删除用户
- 按角色、状态筛选

### 3. 论文管理
- 查看论文列表
- 编辑论文信息
- 删除论文
- 搜索和筛选（按类别、日期、引用数）
- 手动更新论文数据

### 4. 视频管理
- 查看UP主列表
- 查看视频列表
- 搜索和筛选（按UP主、日期、播放量）
- **手动更新B站数据**（包括UP主信息、视频列表、播放量）

### 5. 日志监控
- 登录日志查看
- 访问日志查看
- 操作日志记录
