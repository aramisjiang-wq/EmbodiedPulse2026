## 📝 整理原则

1. **按用途分类**：根据文档的用途和读者对象分类
2. **按类型细分**：优化记录和问题修复按类型进一步细分
3. **保持命名**：文档文件名保持不变，只移动位置
4. **更新索引**：同步更新README.md文档索引
