## 🚀 使用指南

### 启动服务器
```bash
cd robotics_arXiv_daily
python3 app.py
```

### 访问应用
打开浏览器: **http://localhost:5001**

### API 端点

1. **获取统计信息**
   ```
   GET /api/stats
   ```

2. **获取论文列表**
   ```
   GET /api/papers
   ```

3. **搜索论文**
   ```
   GET /api/search?q=关键词&category=类别
   ```

4. **触发抓取**
   ```
   POST /api/fetch
   Body: {"max_results": 10, "config_path": "config.yaml"}
   ```

5. **获取抓取状态**
   ```
   GET /api/fetch-status
   ```

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Flask (Python)
- **数据库**: SQLite + SQLAlchemy
- **API**: RESTful JSON API

## 📝 注意事项

1. **端口**: 使用 5001 端口（避免与 macOS AirPlay 冲突）
2. **数据库**: 首次运行需要执行 `python3 init_database.py`
3. **ArXiv API**: 注意速率限制，建议每次抓取 5-10 篇/类别
4. **备份**: JSON 文件作为备份保留，数据库文件建议定期备份
