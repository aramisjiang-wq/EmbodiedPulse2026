### 首次部署
1. 阅读 `部署教程/从0到1完整部署教程_20251217_v1.0.md`
2. 参考 `配置指南/账号及配置信息清单_20251217_v1.0.md` 配置环境
3. 使用 `部署教程/部署完成检查清单.md` 验证部署

### 配置飞书登录
1. 阅读 `配置指南/飞书OAuth配置指南_20251217_v1.0.md`
2. 按照 `配置指南/飞书登录系统配置指南_20251217_v1.0.md` 配置

2. 使用 `运维管理/运维监控与故障恢复方案_20251217_v1.0.md` 中的恢复脚本
