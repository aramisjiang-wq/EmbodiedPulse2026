# 快速诊断命令

## 步骤1：拉取最新代码

```bash
cd /srv/EmbodiedPulse2026
git pull origin main
```

## 步骤2：运行诊断脚本

```bash
source venv/bin/activate
python3 scripts/check_bilibili_data_status.py
```

## 完整命令（一键执行）

```bash
cd /srv/EmbodiedPulse2026 && git pull origin main && source venv/bin/activate && python3 scripts/check_bilibili_data_status.py
```
