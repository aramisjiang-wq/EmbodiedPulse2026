# Embodied Pulse（具身脉搏） Web Application

这是一个基于 Flask 的 Web 应用，用于浏览和抓取机器人领域的 ArXiv 论文。

## 功能特性

### 具身论文页面
- 📊 **统计信息展示**：显示各类别论文数量和总论文数
- 📚 **论文列表浏览**：33个分类标签，350+检索词，支持切换标签页
- 🔍 **论文详情**：显示论文标题、作者、发布日期、PDF 和代码链接
- 📈 **研究方向活跃度**：实时趋势图、分类视图、子标签视图
- 👥 **活跃作者排行榜**：高产作者追踪、研究方向标注
- 🔄 **自动抓取**：通过 Web 界面触发论文抓取任务
- 📈 **实时状态**：显示抓取进度和状态

### 具身视频Hub页面
- 🎬 **视频发布趋势**：堆叠面积图、热力图视图
- 📊 **播放量统计**：月度播放量、年度总播放量、TOP 5排行
- 👤 **UP主信息**：12个UP主追踪，粉丝数、视频数、播放量统计
- 🎥 **最新视频**：7天内最新视频，按发布时间倒序

### 管理平台
- 👥 **用户管理**：查看、编辑、删除用户
- 📄 **论文管理**：查看、编辑、删除论文，搜索筛选
- 🎬 **视频管理**：查看UP主和视频信息，手动更新B站数据
- 📊 **日志监控**：登录日志、访问日志

## 使用说明

1. **查看论文**：打开网页后会自动加载已有的论文数据
2. **刷新数据**：点击"刷新数据"按钮重新加载论文列表
3. **抓取新论文**：
   - 点击"抓取新论文"按钮
   - 配置抓取参数（每类论文数量、配置文件路径）
   - 点击"开始抓取"
   - 等待抓取完成（会显示进度条）

## API 接口

- `GET /api/papers` - 获取所有论文数据
- `GET /api/stats` - 获取统计信息
- `GET /api/fetch-status` - 获取抓取状态
- `POST /api/fetch` - 触发论文抓取

## 注意事项

- 抓取论文时可能会遇到 ArXiv API 的速率限制（HTTP 429），这是正常现象
- 建议在抓取时设置较小的 `max_results` 值（如 5-10）
- 抓取任务在后台运行，不会阻塞 Web 界面
