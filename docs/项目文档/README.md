## 📊 项目概况

### 项目性质
这是一个**完整的全栈Web应用项目**，包含前端、后端、数据库、数据抓取等完整功能。

### 技术栈
- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **后端**：Flask + SQLAlchemy
- **数据库**：SQLite（可升级到PostgreSQL）
- **数据抓取**：ArXiv API
- **任务调度**：APScheduler

### 核心功能
- ✅ 论文浏览和分类展示
- ✅ 搜索和筛选功能
- ✅ 手动抓取功能（Web界面 + API）
- ✅ 自动定时抓取（APScheduler）
- ✅ 智能去重和增量更新
- ✅ 现代化UI设计
- ✅ 数据库存储和备份

### 数据统计（截至2025-12-08）
- **总论文数**：4,294+ 篇
- **Manipulation**：1,153 篇
- **VLM**：2,165 篇
- **VLA**：492 篇
- **Humanoid**：279 篇
- **Dexterous**：205 篇
