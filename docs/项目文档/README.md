## 📊 项目概况

### 项目性质
**Embodied Pulse**（具身脉搏）是一个面向具身智能、机器人学习、AI研究领域的论文聚合平台。

### 技术栈
- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **后端**：Flask + SQLAlchemy
- **数据库**：SQLite（论文、新闻、招聘、数据集独立数据库）
- **数据抓取**：ArXiv API、Semantic Scholar API、RSS、NewsAPI等
- **任务调度**：APScheduler

### 核心功能
- ✅ 论文浏览和分类展示（10个研究方向）
- ✅ 搜索和筛选功能
- ✅ 手动/自动抓取功能
- ✅ 新闻资讯（24小时更新）
- ✅ 招聘信息（自动抓取）
- ✅ 数据集信息（结构化展示）
- ✅ 互动功能（每日抽签）
