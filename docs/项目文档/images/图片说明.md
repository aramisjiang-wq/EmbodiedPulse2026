## 📝 使用说明

### 在文档中引用图片

Markdown格式：
```markdown
![图片描述](./images/图片文件名)
```

示例：
```markdown
![项目封面](./images/limx封面图.webp)
![GitHub Actions配置](./images/4-ga-2-1.png)
```

### 图片管理规范

1. **命名规范**：
   - 使用描述性的文件名
   - 使用小写字母和连字符
   - 示例：`github-actions-config.png`

2. **格式选择**：
   - 截图：PNG格式（保证清晰度）
   - 照片/封面：JPG或WebP格式（较小文件大小）
   - 图标：SVG格式（矢量图，可缩放）

3. **文件大小**：
   - 尽量控制在1MB以内
   - 大图片可以压缩后使用

4. **添加新图片**：
   - 将图片文件放入本目录
   - 更新本说明文档
   - 在相关文档中引用
