## 🎨 设计规范

### 布局结构

```
┌─────────────────────────────────────────────────┐
│                   Banner                         │
└─────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────┐
│  💬 需求反馈 │ 📚 具身知识库 │ 🛠️ GradMotion │ 🤖 LimX  │
└─────────────────────────────────────────────────┘
```

### 视觉样式

**容器**：
- 白色背景
- 圆角：12px
- 阴影：var(--shadow)
- 底部间距：30px

**按钮**：
- 等宽布局（flex: 1）
- 内边距：16px 20px
- 字体：600粗体，0.95rem
- 右侧分隔线：1px #e5e7eb
- 图标大小：1.1rem

**交互效果**：
1. **悬停**：
   - 背景：淡紫色渐变
   - 文字颜色：#667eea
   - 图标放大：1.15倍
   - 底部渐变线条从左到右展开

2. **底部线条动画**：
   - 高度：3px
   - 渐变：#667eea → #764ba2
   - 初始：scaleX(0)
   - 悬停：scaleX(1)
   - 过渡时间：0.3s

## 💻 技术实现

### HTML结构

```html
<nav class="top-nav">
    <div class="nav-links">
        <a href="https://..." target="_blank" class="nav-link">
            <i class="fas fa-comment-dots"></i>
            <span>需求反馈</span>
        </a>
        <a href="https://..." target="_blank" class="nav-link">
            <i class="fas fa-book"></i>
            <span>具身知识库</span>
        </a>
        <a href="https://..." target="_blank" class="nav-link">
            <i class="fas fa-tools"></i>
            <span>GradMotion算法训练工具箱</span>
        </a>
        <a href="https://..." target="_blank" class="nav-link">
            <i class="fas fa-robot"></i>
            <span>LimX逐际动力</span>
        </a>
    </div>
</nav>
```

### CSS样式

```css
/* 容器 */
.top-nav {
    background: white;
    border-radius: 12px;
    box-shadow: var(--shadow);
    margin-bottom: 30px;
    padding: 0;
    overflow: hidden;
}

/* 链接容器 */
.nav-links {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 0;
}

/* 单个链接 */
.nav-link {
    flex: 1;
    display: flex;
    align-items: center;
    justify-content: center;
    gap: 8px;
    padding: 16px 20px;
    text-decoration: none;
    color: var(--text-primary);
    font-weight: 600;
    font-size: 0.95rem;
    transition: all 0.3s ease;
    border-right: 1px solid #e5e7eb;
    position: relative;
}

.nav-link:last-child {
    border-right: none;
}

/* 图标 */
.nav-link i {
    font-size: 1.1rem;
    transition: transform 0.3s ease;
}

/* 悬停效果 */
.nav-link:hover {
    background: linear-gradient(135deg, #667eea15, #764ba215);
    color: #667eea;
}

.nav-link:hover i {
    transform: scale(1.15);
}

/* 底部线条动画 */
.nav-link::after {
    content: '';
    position: absolute;
    bottom: 0;
    left: 0;
    right: 0;
    height: 3px;
    background: linear-gradient(90deg, #667eea, #764ba2);
    transform: scaleX(0);
    transition: transform 0.3s ease;
}

.nav-link:hover::after {
    transform: scaleX(1);
}
```

## 📱 响应式设计

### 桌面端（> 768px）
- 水平排列，4个按钮等宽
- 右侧分隔线
- 悬停效果完整

### 移动端（< 768px）
```css
@media (max-width: 768px) {
    .nav-links {
        flex-direction: column;
        gap: 0;
    }
    
    .nav-link {
        width: 100%;
        border-right: none;
        border-bottom: 1px solid #e5e7eb;
    }
    
    .nav-link:last-child {
        border-bottom: none;
    }
}
```

**移动端布局**：
- 垂直堆叠
- 全宽按钮
- 底部分隔线
- 保持悬停效果

## 🎯 用户体验

### 视觉层次
```
Banner（紫色渐变，醒目）
    ↓
导航栏（白色，简洁）
    ↓
研究方向（卡片式）
    ↓
统计信息（横幅+网格）
    ↓
论文列表
```

### 交互反馈

1. **视觉反馈**：
   - 悬停时背景色变化
   - 图标放大动画
   - 底部线条展开

2. **操作反馈**：
   - 点击后在新标签页打开
   - 不影响当前页面状态
   - 返回时保持浏览位置

3. **可访问性**：
   - 清晰的图标和文字
   - 足够的点击区域
   - 高对比度配色
